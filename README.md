# High-precision-library
This is a C++ high-precision cauculation library for signed int. It uses thousand-based storage to calculate for efficiency, and it can calculate 2^200000 with in 1s. Most of the operators are overloaded, so you can use it as how you use int and long long.
For detailed instructions, you can read comments at the top of the source code.


# 高精度计算库
这是一个有符号整数的C++高精度计算库。它使用千进制存储以提高效率，可以在1秒内算出2^200000。重载了大部分运算符，你可以像使用int和long long一样使用它。
详细使用说明见源代码顶端的注释。
